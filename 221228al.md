## git 명령어 정리

- 저장소 처음 만들때
  § git init
- 비전을 기곡할 때
  S git add.
  Sgit commit m '커밋메시지'
- 상태 확인할 때
  Sgit status : 1동, 2통 S gt log : 커밋 확인
- 원격저장소 할용하기
  • 원격저장소 설정을 처음 할 때
  S git remote add origin URL
  • push / pull
  $ git push origin master
  S git pull origin master
